# Billetera Virtual 💳

Este proyecto consiste en una simulacion de billetera virtual desarrollada como parte de un ejercicio academico. La aplicacion permite gestionar un saldo inicial mediante operaciones basicas como depositos y retiros, actualizando la informacion en tiempo real en la interfaz.

El objetivo principal es aplicar conceptos fundamentales de desarrollo front-end, incluyendo estructura HTML, estilos con CSS y Bootstrap, y logica con JavaScript y jQuery.

---

## Funcionalidades

- Visualizacion de saldo actual
- Deposito de dinero
- Retiro de dinero
- Validacion de montos ingresados
- Mensajes de alerta dinamicos
- Actualizacion del saldo en tiempo real

---

## Tecnologias utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- jQuery
